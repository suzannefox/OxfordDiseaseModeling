# Modelling the Hagelloch Measles Outbreak of 1861

In 1861 an outbreak of Measles in a small, isolated German village, Hagelloch.   

Dr. Albert Pfeilsticker visited the town each day and recorded the spread of the disease and the complications arising from it. In 1992 Dr. Heike Oesterle augmented and re-analysed the dataset, and it is available in the R surveillance package. 

I used this dataset on the Oxford Short Course [Infectious Disease Modelling: Applied Methods in R](https://lifelong-learning.ox.ac.uk/courses/infectious-disease-modelling-applied-methods-in-r) which covers SEIR compartment modelling of infectious diseases.
